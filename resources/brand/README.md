# EasyPal-Next Brand Assets

Official cartoon-style brand for EasyPal-Next.

## Palette

| Color | Hex | Use |
|-------|-----|-----|
| Royal blue | `#1e4d8c` | Primary, icon background |
| Deep blue | `#1a3a5c` | Headers, web chrome |
| Golden yellow | `#f5c518` | Radio waves, accent text |
| White | `#ffffff` | Antenna, face, primary text |

## Files

| File | Use |
|------|-----|
| `easypal-next-icon.png` | Master app icon (512×512) |
| `easypal-next-icon.svg` | Scalable icon |
| `easypal-next-logo.png` | Horizontal banner (README, docs) |
| `easypal-next-logo.svg` | Scalable horizontal logo |
| `easypal-next.ico` | Windows executable / installer |
| `icon-*.png` | Sized variants for UI |

Web copies live in `src/easypal_next/network/static/brand/`.
